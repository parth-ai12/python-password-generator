# 🔐 Python Password Generator

A simple Python program that generates strong random passwords using letters, numbers, and symbols.

## 🚀 Features
- Generate multiple passwords
- Choose password length
- Uses letters, numbers, and special characters
- Automatically copies the password to clipboard

## 🛠 Built With
- Python
- random module
- string module
- pyperclip

## ▶️ How to Run

1. Clone the repository
2. Run the Python file
